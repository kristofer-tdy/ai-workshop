# AI Workshop

An interactive visual guide to understanding Large Language Models (LLMs), featuring hands-on demonstrations of tokenization, inference, training, and API workflows.

## Overview

This Next.js application provides an educational journey through the core concepts of how LLMs work, presented through interactive visualizations and real-world examples.

## Features

- **Interactive Tokenization Demo** - See how text is broken down into tokens using different methods
- **Inference Visualization** - Experience streaming output and temperature controls in real-time
- **Training Process** - Step through the training lifecycle of an LLM
- **API Flow & Cost Calculator** - Understand API interactions and pricing models
- **RAG (Retrieval-Augmented Generation)** - Explore how LLMs can be enhanced with external knowledge
- **MCP (Model Context Protocol)** - Learn about extending LLM capabilities
- **Challenge System** - Practice with GitHub Copilot and M365 custom agents

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

### Build

```bash
npm run build
npm start
```

## Modes

The application supports two modes:

- **Self-Study Mode** - Navigate freely with sidebar navigation
- **Presentation Mode** - Full-screen slides for workshops and presentations

## Technology Stack

- **Framework**: Next.js 16
- **UI**: React 18, Tailwind CSS
- **Animations**: Framer Motion
- **State Management**: Zustand
- **Language**: TypeScript
